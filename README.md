# App Lista de Clientes
Aplicação desenvolvida para fins didáticos com objetivo de listar contatos de clientes em um CRUD básico na plataforma Android. 

# Especificações

- IDE Android Studio 3.1.3
- Linguagem Java
- Banco de dados SQLite
- API 23: Android 6.0
